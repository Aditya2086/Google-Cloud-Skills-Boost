Task 4: Set up Network Policy
goto editor and in network-policy.yaml add to end

```
---
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: allow-world-to-nginx-ingress
  namespace: default
spec:
  podSelector:
    matchLabels:
      app: nginx-ingress
  policyTypes:
  - Ingress
  ingress:
  - {}
```
```
kubectl apply -f network-policy.yaml
```
